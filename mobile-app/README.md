# Royal SMM Mobile App Wrapper

This folder contains a simple mobile app shell for the existing Royal SMM web app.

## Quick start
1. Replace the placeholder URL in mobile-app/www/index.html with your live site URL.
2. Build with PhoneGap / Cordova or open the folder in a mobile wrapper tool.
3. Install the app on Android/iOS and it will open your web app inside a mobile-friendly container.

## Notes
- This is a lightweight wrapper, not a full native rebuild.
- The existing PHP site remains the app backend.
